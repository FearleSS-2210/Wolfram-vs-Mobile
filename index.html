Wolfram vs MobileNet

We did a test and found out that which is more accurate, Wolfram or MobileNet

Test 1 - Water Bottle

Wolfram = Bottled Water ; MobileNet = Microphone

Test 2 - Ruler

Wolfram = Rule ; MobileNet = Shovel

Test 3 - Pen

Wolfram = Refractometer ; MobileNet = File

This shows that none of them are exactly accurate but Wolfram was more closer to the right answers than MobileNet.
